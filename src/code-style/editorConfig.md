# editorConfig

## 作用

统一不同的操作系统，编辑器写出的代码风格。

## 使用

1. 安装扩展 `editorconfig for vs code`。

1. 配置文件 `.editorconfig`

    ```
    root = true

    [*]

    end_of_line = LF
    charset = utf-8
    max_line_length = 100
    indent_size = 2
    indent_style = space
    insert_final_newline = true
    trim_trailing_whitespace = true
    ```
